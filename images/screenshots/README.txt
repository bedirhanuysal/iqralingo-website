IqraLingo - Ekran Goruntuleri (Screenshots)
=============================================

Bu klasore asagidaki ekran goruntuleri PNG formatinda eklenmelidir:

1. dashboard.png   - Ana Sayfa ekrani (namaz vakitleri, ozellikler)
2. lessons.png     - Ders icerigi ekrani (uniteler, basla butonu)
3. quiz.png        - Mini Test ekrani (kelime tanima testi)
4. quran.png       - Kuran ekrani (sure listesi)
5. stats.png       - Istatistik ekrani (haftalik aktivite, detayli istatistikler)

Onerillen boyut: 1080x2340 piksel (veya telefonunuzun ekran boyutu)
Format: PNG
